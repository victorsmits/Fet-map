# Fet-map

## JSON
```
{
    "Name" : STRING,
    "Player" : STRING,
    "x": Number,
    "y": Number,
    "Team": String
}
```

- Name : Player steam name.
- Player : In game player name.
- x : x position in game.
- y : y position in game.
- Team : truck brand.
